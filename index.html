2

실습 단계별 가이드

1

교사 계정으로 로그인

hi.goe.go.kr 접속 → [교사] 클릭 → 아이디/비밀번호 입력

💡 아이디 저장 + 비밀번호 자동저장 꼭 체크!
2

AI 서논술형 평가 메뉴 진입

상단 메뉴 [AI 서·논술형 평가] 클릭 → [+ 수업 추가] 버튼

3

수업 추가 및 평가 대상 설정

학기/학년/과목/수업명 입력 → 평가 대상 탭에서 학생 추가

💡 오늘 실습은 아래 가상 학생 명단 파일을 사용하세요
4

평가 설계 — 루브릭 입력

평가명 입력 → 성취기준 선택 → 채점기준(루브릭) 작성
채점요소 / 배점 / 세부기준 3가지를 채워주세요

💡 루브릭이 명확할수록 AI 채점이 정확해집니다!
5

샘플 답안 업로드

학생 과제물 탭 → PDF 파일 업로드 → 페이지 분할 설정

💡 아래에서 샘플 답안 PDF를 받아 사용하세요
6

AI 채점 실행

AI 채점 탭 → 채점 대상 학생 선택 → [AI 채점 실행] 버튼 클릭
채점 완료까지 1~3분 소요

7

결과 확인 및 피드백 수정

AI 채점 근거 확인 → 피드백 초안(잘한 점/보완할 점) 검토 → 교사 직접 피드백 수정 후 저장

💡 AI 초안을 그대로 쓰지 말고 교사의 눈으로 한 번 더 확인해요
3

자주 막히는 포인트

Q 로그인이 안 돼요! ▼
비밀번호를 5번 잘못 입력하면 5분간 로그인이 제한됩니다. 잠시 기다렸다가 다시 시도해 보세요. 그래도 안 되면 찰리쌤 불러주세요 😅
Q 루브릭을 어떻게 작성해야 하나요? ▼
채점요소(평가하고 싶은 항목) + 배점(점수) + 세부기준(각 점수에 해당하는 기준 설명) 3가지를 채우면 됩니다. 오늘 실습용 루브릭 예시 파일을 참고해 보세요!
Q 답안 업로드가 안 돼요! ▼
PDF 형식만 업로드 가능합니다. 오늘 실습용 샘플 답안은 이미 PDF로 준비되어 있어요. 혹시 직접 만드신 파일이라면 PDF로 변환 후 업로드해 주세요.
Q AI 채점 결과가 이상한 것 같아요! ▼
루브릭의 세부기준이 모호하면 AI도 흔들립니다. 채점 결과는 어디까지나 초안이에요. 교사가 직접 수정할 수 있으니 이상하다 싶으면 수정하면 됩니다. AI 채점의 최종 권한은 항상 교사에게 있어요!
Q AI 논술 진단이 우리 교과는 안 되나요? ▼
현재 AI 논술진단은 국어, 영어, 사회, 과학 교과에서 지원됩니다. 그 외 교과도 교사가 직접 루브릭을 설계하면 서술형 평가에 활용할 수 있어요. 추후 전 교과로 확대될 예정입니다.
